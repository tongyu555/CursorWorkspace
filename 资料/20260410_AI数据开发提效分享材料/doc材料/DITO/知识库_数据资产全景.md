## 4. 核心业务指标口径

> 以下指标口径从生产ETL SQL中提取，是最权威的业务计算逻辑。

### 4.1 无线KPI指标

| 指标名称 | 计算公式 | 来源SQL |
|----------|----------|---------|
| PRB利用率(下行) | `sum(puschprbtotmeandl) / sum(prb_dl_avail)` | `dm_wirelesskpi_station_index_d` |
| RRC接入成功率 | `(sum(rrc_succconnestab_ue)+sum(rrc_succconnestab_net)) / (sum(rrc_attconnestab_ue)+sum(rrc_attconnestab_net))` | `dm_wirelesskpi_station_index_d` |
| 寻呼拥塞率 | `sum(paging_discard_num) / sum(paging_s1_rx)` | `dm_wirelesskpi_station_index_d` |
| UE上下文掉线率 | `sum(uecntx_abnormrel) / (sum(uecntx_abnormrel)+sum(uecntx_normrel))` | `dm_wirelesskpi_station_index_d` |
| E-RAB掉线率 | `sum(erab_abnormrel) / (sum(erab_abnormrel)+sum(erab_normrel))` | `dm_wirelesskpi_station_index_d` |
| 小区可用率 | `1 - sum(cell_unserv_time) / sum(cell_unserv_time/(1-cell_servtime_rate))` | `dm_wirelesskpi_station_index_d` |
| 频谱效率(下行) | `sum(pdcp_dl_bit) / sum(prb_dl_dtch)` | `aggr_lte_pm_cel_api_d` |
| 频谱效率(上行) | `sum(pdcp_ul_bit) / sum(prb_ul_dtch)` | `aggr_lte_pm_cel_api_d` |

### 4.2 MR覆盖指标

| 指标名称 | 计算公式 | 来源SQL |
|----------|----------|---------|
| MR RSRP≥-105覆盖率 | `sum(mr_105_cover_rate * mr_num) / sum(mr_num)` | `dm_wirelesskpi_station_index_d` |
| MR RSRP≥-110覆盖率 | `sum(mr_110_cover_rate * mr_num) / sum(mr_num)` | `dm_wirelesskpi_station_index_d` |
| 弱覆盖小区标志 | 连续3天：`sum(cellgrid_rsrp_cnt)>500` 且弱覆盖MR占比>0.5 | `aggr_grid_outdoor_cell_d` |
| 重叠覆盖标志 | `overlap_mrcount/total > 0.05` 且 `avg_rsrp >= -95` 且 `overlap_mrcount >= 2` | `aggr_nr_mr_cell_kpi_h` |
| 过覆盖标志 | `overshoot_mrcount/total > 0.6` 且 `avg_rsrp >= -95` 且 `overshoot_mrcount >= 2` | `aggr_nr_mr_cell_kpi_h` |
| Mod30干扰标志 | `mod30interfere_mrcount/total > 0.05` 且 `avg_rsrp >= -95` | `aggr_nr_mr_cell_kpi_h` |

### 4.3 5G NR性能指标

| 指标名称 | 计算公式 | 来源SQL |
|----------|----------|---------|

## 8. 取数SQL编写指南

### 8.1 基本原则

1. **必须带分区条件**：所有查询必须指定分区字段（p_date/p_hour/p_provincecode等）
2. **禁止SELECT ***：必须明确列出所需字段
3. **优先使用DM/AGGR层**：取数优先从应用层（DM）或汇总层（AGGR）取数，避免直接查ODS
4. **指标口径对齐**：计算业务指标时，优先参考`sql_code/`中的ETL逻辑
5. **Hive SQL方言**：使用Hive标准函数（nvl/coalesce/date_format），禁止MySQL/PG函数

### 8.2 常用取数模式

**模式一：单表直取（简单统计）**
```sql
SELECT
    day
    ,provincecode
    ,citycode
    ,cellkey
    ,rrc_succconnestab_rate
    ,erab_drop_rate
FROM
    dm_wirelesskpi_cell_index_d
WHERE
    p_date = '${p_date}'
    AND p_provincecode = ${p_provincecode}
```

**模式二：多表关联（指标拼接）**
```sql
SELECT
    A1.cellkey
    ,A1.cellname
    ,A1.rrc_succconnestab_rate --RRC接入成功率
    ,A2.avg_rsrp              --MR平均RSRP
    ,A2.mr_105_cover_rate      --RSRP≥-105覆盖率
FROM
    (SELECT
        cellkey
        ,cellname
        ,rrc_succconnestab_rate
    FROM
        dm_wirelesskpi_cell_index_d
    WHERE
        p_date = '${p_date}'
        AND p_provincecode = ${p_provincecode}
    )A1
    LEFT JOIN
    (SELECT
        cellkey
        ,avg_rsrp
        ,mr_105_cover_rate
    FROM
        dm_wirelesscover_cell_detail_d
    WHERE
        p_date = '${p_date}'
        AND p_provincecode = ${p_provincecode}
    )A2
    ON A1.cellkey = A2.cellkey
```
