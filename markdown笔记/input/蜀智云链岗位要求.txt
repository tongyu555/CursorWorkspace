# 资深大数据开发/数据治理工程师  岗位要求
## 岗位职责
1. 主导多源异构数据的采集、清洗、转换和加载(ETL/ELT)架构设计与实施，实现业务系统与数据中台的高效对接，保障物流核心业务数据(运输、仓储、配送等)的实时性和准确性，支撑实时数据处理场景。
2. 建立完整的数据质量监控体系，制定数据质量规则和校验标准，开发数据质量检测工具和自动化脚本，持续提升数据准确性、完整性和一致性，推动数据质量问题的闭环管理。
3. 主导数据标准、数据模型、元数据管理等数据治理规范的制定与落地，设计并实施数据仓库分层架构(ODS/DWD/DWS/ADS)，维护数据字典和元数据仓库，建立数据血缘关系，推动主数据管理(MDM)体系建设。
4. 基于Hadoop/Spark/Flink生态和公司自研数据中台，主导数据治理相关功能模块的架构设计与开发，优化大数据处理任务性能，参与数据中台整体架构设计与技术选型，推动数据治理工具和平台的落地应用。
5. 与业务部门、产品团队、开发团队紧密协作，理解业务需求，设计并推动数据治理解决方案落地，为数据分析和决策应用提供高质量的数据资产，建立数据治理最佳实践。

## 任职资格
1. 教育背景：本科及以上学历，计算机科学、软件工程、数据科学、信息管理或相关专业优先。
2. 工作经验：3-5年数据工程、数据治理或相关领域工作经验，具备完整的数据治理项目实施经验，有物流、供应链、电商等行业经验者优先。
3. 技术能力：
- 数据湖技术：深入理解Data Lakehouse架构，精通Apache Iceberg 或 Apache Paimon至少一种数据湖格式，有实际的生产环境落地经验（如解决CDC数据入湖、Schema Evolution、小文件合并等问题）。
- OLAP引擎：熟练掌握Apache Doris（或StarRocks、ClickHouse），具备基于Doris构建实时数仓、物化视图设计及查询性能调优的经验。
- 计算引擎：精通Flink 或 Spark引擎，熟悉Flink + Paimon/Iceberg的流式数仓开发模式。
- 基础能力：扎实的SQL 和 Java/Scala/Python编程能力，熟悉Hadoop生态（HDFS/Hive）并具备基于Hadoop环境进行数据开发的能力。
- 数据治理组件：熟悉数据质量管理，掌握数据治理工具(Collibra、DataHub、Atlas)、BI工具者优先。
4. 业务理解：理解物流行业业务流程和数据特点，能够将业务需求转化为技术方案，具备良好的数据敏感度，了解供应链、运输、仓储数据的治理场景。
5. 软技能：优秀的问题分析和解决能力，良好的沟通协调能力，能够与技术团队和业务团队高效协作，具备强烈的责任心和主动性。

（二）大数据 / 数仓 / 数据治理专属业务（完全匹配你投递的资深岗 JD）
1. 底层数据链路业务（ETL/ELT 湖仓一体）
承载集团四大核心业务数据源的全链路采集：
物流业务：蜀好运货车运力、北斗定位、运输履约、司机行为、高速通行埋点日志；
仓储供应链：蜀智云仓出入库、库存、园区监控、货物调拨；
集采大宗：蜀智云商砂石 / 钢材 / 煤炭采购、订单、供应商交易 CDC 数据；
业财一体化：业务单据、结算、财务对账多源 MySQL 业务库。
技术方案：Flink CDC+DataX 完成实时 / 离线双模式入湖，基于Iceberg/Paimon 数据湖做统一存储，解决 CDC 增量、Schema 变更、小文件治理，支撑流批一体实时 / 离线计算。
2. 数据仓库分层建设（ODS/DWD/DWS/ADS）
完整四层湖仓分层落地，面向两类输出场景：
1）经营分析报表：集团月度营收、运力周转、仓储库存、集采交易 T+1 离线指标；
2）政府实时监测大屏：全省物流运力预警、物资调度、交通产业实时指标，基于 Apache Doris 构建实时数仓、物化视图支撑低延迟查询。
团队核心工作：维度建模、缓慢变化维处理、主题域（物流 / 仓储 / 集采）指标体系统一。
3. 全链路数据治理（岗位核心权重最高）
这是该岗位核心考核模块，也是公司长期重点投入板块：
数据质量闭环：搭建五大校验规则（完整性 / 唯一性 / 准确性 / 一致性 / 及时性），开发自动化检测脚本，实时 + 离线双监控，形成问题上报 - 修复 - 复核闭环；
元数据 & 血缘 & 主数据 MDM：自研元数据平台，维护全量数据字典、自动解析数据血缘，统一物流、供应商、车辆、仓库主数据标准；
数据标准落地：制定集团统一指标口径、数据分层规范、数据安全脱敏规则，推动全业务线对齐；
治理平台自研迭代：基于 Hadoop/Spark/Flink 生态，开发数据质量、元数据管理模块，替代第三方工具（Atlas/DataHub），适配国企审计、数据安全要求。
4. 数据中台协同工作
对接业务、产品、集团财务部、交通政府部门，输出数据解决方案，输出审计验收材料、数据资产报告，搭建供应链数据最佳实践，支撑风控、运力预测、产业决策分析。
标配技术栈（JD 硬性要求，全部生产落地）
数据湖：Iceberg/Paimon；OLAP：Apache Doris；计算引擎：Spark/Flink；同步：DataX/Flink CDC；调度：自研调度平台；治理：自研元数据 / 质量平台；配套 Hive/HDFS/Kafka/Redis。
